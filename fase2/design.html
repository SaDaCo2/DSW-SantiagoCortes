Etiquetas semánticas

La semántica es la ciencia de comprender algunos signos,secuencias de símbolos y otras designaciones convencionales. El código HTML semántico es un diseño con el uso correcto de las etiquetas HTML, usándolos por su objetivo pronosticado, ya que fueron creados por los desarrolladores del idioma HTML y los estandares web, mejorando la disponibilidad de información en el sitio, haciendolo más rápido y menos costoso. Es de gran importancia para agentes alternativos como: navegadores de voz, dispositivos móviles, dispositivos de impresión, etc.

Romance, V. A. Correspondencia en la semántica del código HTML con ejemplos. Codificando qué código semántico. https://neonkaraoke.ru/es/internet/rassuzhdeniya-o-semantike-koda-html-s-primerami-kodirovanie-chto-takoe/

CSS Grid

Un grid posibilita la visualización, el manejo, ordenación y operación de una fuente de datos. Como resultado de este grid se obtendrá una tabla donde cada fila un registro del origen de datos y cada columna representará un campo. También a parte de generar columnas y filas para presentar los valores de un origen de datos, este grid además nos permitirá el calculo y la operación del resto de campos y valores de nuestro gird y de esta manera presentar el resultado de operaciones en tiempo de ejecución.

García Salinero, M. (2014). Grid de datos desarrollo con AngularJS utilizando el patrón Modelo Vista-Controlador. https://ebuah.uah.es/xmlui/handle/10017/20841
